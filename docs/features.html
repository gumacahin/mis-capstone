F-00: User registration

Priority: Essential
Effort: Days
Risk: Safe
Functional area(s): Getting Started
Use case(s): UC-00 UC-01 UC-02 UC-03
Description: Visitors can come to the System and register themselves. They must provide the following information:
  • email address
  • name
  • password
  • password confirmation

Precise Details:

  • email must be unique (must not be registered with the System before)
  • email address will be verified by sending the user's initial password there
  • name must not be empty
  • password must not be empty
  • password confirmation must not be empty and must be identical to password
  • Password must be at least eight characters long and must contain at least one uppercase character, at least one lowercase character, at least one digit and at least one special character
  • leading and trailing spaces are stripped from all fields
